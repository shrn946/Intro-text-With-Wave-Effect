# Intro Text With Wave Effect

**Version:** 1.0  
**Author:** Your Name

## Description

Integrates GSAP text effect scripts into your WordPress site. Add engaging wave effects to your introductory text with ease.

### Usage
Use the following shortcode to implement the wave effect in your content:
```[custom_splash_section text="BEST <span>TRENDS !</span>" font_size="100px"]```

## Installation

1. Upload the 'intro-text-wave-effect' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
